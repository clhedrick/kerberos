Host Management

This page lists host that you manage. That means that you can create key tables for them, and thus you can set up Kerberos on them.

Add Host

This adds hosts into our Kerberos system.

  • The hostname must already be registered in DNS. That is normally done when the IP address is assigned.
  • The host must not already be in our system. (This is to prevent people from taking over ownership of someone else's system.
  • You must have permission to add hosts. All faculty do. They can request us to authorize others.
