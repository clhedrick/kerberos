NOTE: This application accepts Kerberos single-signon. If you use that you won't need to type your username and password. This should work on any computer that supports the Computer Science Kerberos system. That includes all computers run by Computer Science staff. You can set it up on your own computer as well. See Integrating your computer into Kerberos.

If your computer supports our Kerberos, see Configure Browsers for Spnego Negotiaton You can either whitelist https://services.cs.rutgers.edu or depending upon the browser, *.cs.rutgers.edu. Note that Chrome on the Mac is like Linux in accepting a JSON file in /etc/opt/chrome/policies/managed
