NOTE: This application accepts Kerberos single-signon (technically called "SPNEGO") if you are using a browser on a system where login uses Computer Science Kerberos. (This is true for all Linux systems run by Computer Science staff. You can configure our own Mac or Linux systems that way as well.) .

If you use single-signon, you won't need to type your username and password to get to most screens.
